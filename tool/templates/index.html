① LLM Token Expenditure Index (Silicon Data)

市场为每百万 LLM token 支付的使用量加权均价 — AI 边际付费意愿。无公开 API，由 Claude 从 @Silicon_Data 公开发布人工跟踪（空心点为估算值）；在 Claude 会话中运行 /update-silicon-data 可补充最新值。数据截至 —。

② CNN Fear & Greed

CNN 综合 7 个子指标（动量、强弱、宽度、期权、垃圾债需求、避险需求、波动率）合成的 0-100 情绪温度计： <25 极度恐慌 / 25-45 恐慌 / 45-55 中性 / 55-75 贪婪 / >75 极度贪婪。极端读数常用作反向信号——极恐近底、极贪近顶。 源：CNN dataviz API（准实时，盘中变动）。

③ VIX (Volatility Index)

标普 500 期权隐含的未来 30 天年化波动率，俗称"恐慌指数"——市场为对冲下跌愿付的保险价格。 <15 平静 / 15-20 正常 / 20-30 紧张 / >30 恐慌。注意：本图为 CBOE 现货 VIX 指数日收盘（非 /VX 期货，期货价含升贴水会有差异）。 源：CBOE 官方 EOD 数据 → Yahoo → FRED 三级链。

④ Market Breadth — % above 50DMA

纳指 100 / 标普 500 中股价站上 50 日均线的成分股比例——衡量上涨是"普涨"还是少数权重股拉指数。 >70% 强势普涨 / 50-70% 健康 / 30-50% 偏弱（结构性行情）/ <30% 超卖。指数新高但宽度走低 = 顶背离警示。 源：StockCharts $NDXA50R / $SPXA50R（日收盘）。

⑤ AAII Investor Sentiment

美国个人投资者协会每周问卷：未来 6 个月看涨/中性/看跌占比。Bull-Bear 净值常态约 +6； >+20 散户过热（反向偏空）/ <-20 散户过冷（反向偏多，历史上常对应阶段性底部附近）。散户情绪是典型反向指标。 源：aaii.com 官方周度 XLS（每周四美东发布）。

⑥ Put/Call Ratio (CBOE Equity)

CBOE 个股期权认沽/认购成交量比（equity-only，剔除指数对冲盘）。 <0.6 极度乐观（自满风险）/ 0.6-0.8 乐观 / 0.8-1.0 中性 / >1.0 避险情绪重（反向看常近底）。 源：StockCharts $CPCE（CBOE 授权数据，日收盘）。
